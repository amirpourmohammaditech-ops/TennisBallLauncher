# Arduino Tennis Ball Launcher

An Arduino-powered dual-wheel tennis ball launcher designed for tennis training. The launcher uses two high-torque DC motors driven by BTS7960 motor drivers, allowing adjustable launch speed through PWM control.

## Features
- Dual-wheel launch system
- Arduino Uno controller
- BTS7960 motor drivers
- Adjustable launch speed
- Modular design for future upgrades

## Hardware
- Arduino Uno
- 2× BTS7960 motor drivers
- 2× 24V DC motors
- 24V power supply
- 10k potentiometer (planned)

## Future Improvements
- Adjustable launch angle
- Oscillation
- Ball feed mechanism
- Remote control
